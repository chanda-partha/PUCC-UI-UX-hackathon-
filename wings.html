Functional Specializations

Explore Our Specialized Wings

Choose a domain to collaborate with mentors, deploy projects, and develop production-grade skills.

CP WING

Competitive Programming

"Nurture problem-solving and prepare members for national and international contests."

Key Activities

  • • Weekly mock ICPC contests & editor sessions
  • • Intensive training camps on number theory
  • • Mentorship from top Codeforces and CodeChef rankers

Skills Gained

Dynamic Programming, Graph algorithms, Complex math optimization, Time complexity diagnostics.

Technologies

C++ Java STL Python
SE WING

Software Engineering

"Equip members to design, develop, deploy modern, user-centric software to industry standards."

Key Activities

  • • Open-source contributions & team git sprints
  • • Full-stack web application deployment sprints
  • • CI/CD pipeline and cloud setup workshops

Skills Gained

REST API architecture, Database schema design, UI/UX aesthetics, Testing & code refactoring.

Technologies

React.js Node.js Docker Git
NETWORKING WING

Linux & Networking

"Build foundations in Linux, networking, and security for IT infrastructure innovation."

Key Activities

  • • Shell scripting challenges & automation
  • • Routing/switching simulation in virtual environments
  • • Server setup (Nginx, Apache, DNS) labs

Skills Gained

Terminal proficiency, Server hardening, Subnet planning, Bash automation, Cloud infrastructure.

Technologies

Ubuntu/RHEL Bash Cisco Packet Tracer AWS
AI RESEARCH WING

Deep Neural Research

"Advance AI, ML, and deep learning through cutting-edge models and real-world use."

Key Activities

  • • Neural network training and parameter tuning
  • • Research paper discussions on CNNs and LLMs
  • • Computer vision datasets optimization sprints

Skills Gained

Data preprocessing, Model evaluation, Neural networks setup, Computer vision algorithm deployment.

Technologies

Python TensorFlow PyTorch Keras
R&D WING

Research & Development

Skills Gained

Technologies

Latex CAD Tools SolidWorks Python

Career Guidance

LinkedIn LaTeX CVs ATS Scanners
